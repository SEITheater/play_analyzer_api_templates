Drag and drop PML files onto this web page in order to use their contents and the PlayAnalyzer.com server to create interactive web displays.

JQuery and Dropzone are used to handle the file upload process. The file_uploader.js file contains a global variable called globalFileList that contains all files dragged onto the page during a session. make_api_request.js contains a function called makePostRequest that will make requests to the server for you and return the response to a specified callback.
